## Description

Boilerplate application with Nest.js & TypeORM.

## Pre-requirements

- node.js (tested on v. 12.6.0)
- docker
- docker-compose

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start:local

# test
$ npm run test:integration
```
